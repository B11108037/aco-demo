Core idea of ACO on a grid

This version removes fixed routes. Each ant must move through the grid one cell at a time. When ants find food, their successful path receives pheromone. Later ants combine pheromone strength and distance-to-goal information when choosing the next grid cell. The yellow cells are not calculated by BFS; they are the best simplified path discovered by ants so far.

1. ExploreAnts search the grid by moving cell by cell.
2. Find foodSuccessful ants remember the path they took.
3. DepositThe successful path receives pheromone.
4. ImproveIf a later ant discovers a shorter path, the best path updates.
Weight(next cell) ∝ (pheromone ^ α) × ((1 / distance to target) ^ β)
