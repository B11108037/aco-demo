Core idea of ACO on a grid

This version removes fixed routes. Each ant must move through the grid one cell at a time. When ants find food, their successful path receives pheromone. Later ants combine pheromone strength and distance-to-goal information when choosing the next grid cell. Over time, pheromone forms a visible path from the nest to the food.

1. ExploreAnts search the grid by moving cell by cell.
2. Find foodSuccessful ants remember the path they took.
3. DepositThe successful path receives pheromone.
4. FollowLater ants are attracted to stronger pheromone cells.
Weight(next cell) ∝ (pheromone ^ α) × ((1 / distance to target) ^ β)
